# MultiLayer-Perceptron-Xor

1. Instala o Python (se ainda não tiver).
2. Abre o VS Code e seleciona o interpretador global do Python (Ctrl+Shift+P → Python: Select Interpreter).
3. No terminal do VS Code, instala as bibliotecas necessárias:
   python -m pip install numpy matplotlib
4. Depois é só abrir os arquivos:
   - ativacoes.py → funções de ativação
   - mlp.py → estrutura da rede
   - experimento_xor.py → treinamento do XOR
   - experimento_extra.py → experimentos A, B e C
   - teste.py → pra rodar e ver os gráficos
